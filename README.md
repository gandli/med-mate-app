# MedMate (伴药)
慢性病患者及家属的本地化健康管理工具

## 项目概述
专注用药管理、症状追踪、复诊提醒的微信小程序

## 核心功能
- 用药管理（多药协同、中西药覆盖）
- 症状追踪
- 复诊提醒
- 家属协同
- 安全检测（药物相互作用）

## 技术栈
- 微信小程序
- Supabase
- 大模型AI解读

## 核心价值

大厂做"查一下"，我们做"管起来"：

1. **多药全周期管理**（不是单次查药）
2. **中西药全覆盖**（中药饮片/中成药/院内制剂）
3. **家属协同**（子女远程监督父母吃药）
4. **安全检测**（多药相互作用+中西药冲突+肝肾负担）
