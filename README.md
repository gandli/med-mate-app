# MedMate (伴药)
慢性病患者及家属的本地化健康管理工具

## 项目概述
专注用药管理、症状追踪、复诊提醒的微信小程序

## 核心功能
- 用药管理（多药协同、中西药覆盖）
- 症状追踪
- 复诊提醒
- 家属协同
- 安全检测（药物相互作用）

## 技术栈
- 微信小程序
- Supabase
- 大模型AI解读
